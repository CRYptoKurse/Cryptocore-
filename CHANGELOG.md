## [1.0.0] - 2024-01-15

### Добавлено
- Полная реализация криптографической библиотеки:
  - AES в режимах ECB, CBC, CFB, OFB, CTR
  - Аутентифицированное шифрование GCM
  - Режим Encrypt-then-MAC (ETM)
  - Хеш-функции SHA-256 и SHA3-256
  - HMAC-SHA256
  - Выработка ключей PBKDF2
  - HKDF иерархия ключей
  - Криптографический ГСЧ

- Интерфейс командной строки:
  - Шифрование/дешифрование файлов
  - Вычисление и проверка хешей
  - HMAC генерация и проверка
  - Выработка ключей из паролей

- Комплексная документация:
  - Документация API
  - Руководство пользователя
  - Руководство разработчика

- Набор тестов:
  - Модульные тесты
  - Интеграционные тесты
  - Тестовые векторы NIST/RFC

### Безопасность
- Все реализации соответствуют стандартам
- Безопасное управление памятью
- Проверка входных данных
- Постоянные по времени операции
- Аутентификация перед дешифрованием

### Производительность
- Оптимизированные реализации
- Поддержка потоковой обработки
- Минимальное использование памяти
