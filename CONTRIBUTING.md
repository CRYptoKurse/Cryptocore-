# Участие в разработке CryptoCore

Благодарим за интерес к участию в разработке CryptoCore! Этот документ содержит рекомендации и инструкции для участников.

## Кодекс поведения

Пожалуйста, прочитайте и соблюдайте наш [Кодекс поведения](CODE_OF_CONDUCT.md).

## Как я могу участвовать?

### Сообщение об ошибках

Перед созданием отчета проверьте существующие issues.

**Как сообщить об ошибке:**
1. Используйте шаблон отчета
2. Опишите шаги для воспроизведения
3. Укажите информацию о системе
4. Включите сообщения об ошибках
5. Добавьте минимальный воспроизводимый пример

### Предложение улучшений

**Как предложить улучшение:**
1. Проверьте существующие предложения
2. Используйте шаблон feature request
3. Объясните пользу улучшения
4. Приведите примеры использования

### Pull Requests

**Процесс:**
1. Сделайте форк репозитория
2. Создайте ветку для новой функции
3. Внесите изменения
4. Добавьте тесты
5. Обновите документацию
6. Отправьте PR

**Требования к PR:**
- Соблюдение стандартов кода
- Тесты для новой функциональности
- Обновленная документация
- Одно изменение на PR
